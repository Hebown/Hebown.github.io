数据分析/处理：#

  • data management:
    • data profiling:
      • data overview:
        • data domain:数据属于什么领域；
        • 数据规模；
        • 数据类型；
        • 数据范围；
      • data quality:
        • 缺失信息；
        • 非法信息；
        • 冗余信息；
        • 同质信息：比如zju和浙江大学本质上是一样的。
  • data wrangling（整理）:对数据做出若干操作，比如排序/提取/次数据…
    • 有一个比较抽象的join操作，分为inner，right/left，full，其实就是将两个不同表中的相同元素组合起来成为一个表，不过有不同的组合策略；
  • data modeling:
    • 没讲；
